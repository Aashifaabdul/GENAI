# News Research Tool!!!
![image](https://github.com/user-attachments/assets/cfa87f73-1325-4176-9049-9e9286412b7f)


This project is a lightweight AI-powered news Q&A app that extracts answers from articles based on user queries.
## 🧠 It uses:
- 🔗 **LangChain** to handle LLM chaining and question-answering logic  
- 🤖 **Ollama** (local LLM server) for processing and inference  
- 🖥️ **Streamlit** for an interactive and clean frontend demo  
## 💡 Features
- ✅ Accepts multiple news article URLs  
- 🔍 Extracts relevant answers based on user questions  
- 📎 Provides direct sources from which the answer was derived  
- 🛡️ Fully local inference with Ollama – no external API calls required!
